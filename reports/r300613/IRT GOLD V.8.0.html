System A — EMA Bounce BUY

MSZ Zone Engine

ตรวจหาโซน SUPPLY / DEMAND จากการอิ่มตัวของ Momentum พร้อมให้คะแนนความแข็งแกร่ง 0-100

💡 หลักการสำคัญ ทุกสัญญาณถูกกรองด้วย Choppiness Index (CI) — หากตลาด Sideway (CI > 61.8) สัญญาณจะไม่ fire แม้เงื่อนไขอื่นครบ เพื่อหลีกเลี่ยงการเทรดในตลาดที่ไม่มีทิศทาง
SECTION 02
สัญญาณ IRT B และ IRT S
ป้าย IRT B และ IRT S จะปรากฏหลังจาก Trade ปิดเท่านั้น (ประวัติ)
IRT B

IRT B — สัญญาณ BUY

ป้ายสีเขียวปรากฏ ใต้แท่งเทียน ณ จุดที่เข้าเทรด BUY

เงื่อนไข: Supertrend ขาขึ้น + CI Trending + EMA Fast > EMA Slow + ราคา Bounce จาก EMA

IRT S

IRT S — สัญญาณ SELL

ป้ายสีแดงปรากฏ เหนือแท่งเทียน ณ จุดที่เข้าเทรด SELL

มาจาก S1 (ST พลิก + VWAP) หรือ S2 (Double Top + Neckline break)

📌 ทำไมป้ายถึงโชว์หลัง Trade ปิด? IRT B / IRT S แสดงเฉพาะ Trade ที่ ปิดไปแล้ว (TP หรือ SL โดน) เพื่อให้เห็น History ที่ชัดเจน — Trade ที่กำลังเปิดอยู่จะไม่มีป้าย เพื่อไม่ให้ชาร์ตรก

กล่อง SL/TP

องค์ประกอบ สี ความหมาย
กล่องสีเขียว (บน)เขียวโซน Take Profit — TP ราคาเท่าไหร่
กล่องสีแดง (ล่าง)แดงโซน Stop Loss — SL ราคาเท่าไหร่
เส้น Entryสีขาวราคาที่เข้าเทรด
กล่องจางลง (SL โดน)โปร่งมากขึ้นTrade นั้นปิดด้วย Stop Loss

ระบบสี ✓ / ✗ และการอ่านสัญญาณ
แต่ละเงื่อนไขแสดงสีตามทิศทางและความสอดคล้องกับ Dominant Direction
✓

✓ สีเขียว — Align กับ BUY Dominant

เงื่อนไขนี้ชี้ ขาขึ้น และสอดคล้องกับทิศทางหลักที่ Score BUY ชนะ — สัญญาณยืนยันฝั่ง BUY

✓

✓ สีแดง — Align กับ SELL Dominant

เงื่อนไขนี้ชี้ ขาลง และสอดคล้องกับทิศทางหลักที่ Score SELL ชนะ — สัญญาณยืนยันฝั่ง SELL

✗

✗ สีส้ม — ขัดแย้งกับ Dominant Direction

เงื่อนไขนี้ชี้ ทิศทางตรงข้าม กับ Score ส่วนใหญ่ — เป็น Signal ที่ขัดแย้ง ควรระวัง
CI Sideway จะเป็นส้มเสมอ เพราะไม่มีทิศทาง

ตัวอย่าง Perfect BUY (4/4 ทุกตัวเขียว)

✓ Supertrend UP    ← ST ขึ้น
✓ EMA 9 > EMA 21  ← EMA bullish
✓ RSI > 50        ← Momentum บวก
✓ H4 Trend UP    ← H4 ขึ้น
✓ CI Uptrend (42) ← ตลาด Trending
✓ VWAP Uptrend   ← ราคาเหนือ VWAP

ตัวอย่าง SELL Dominant (3/4) — มีขัดแย้ง 1 ตัว

✗ Supertrend UP  ← ST ขึ้น แต่ขัด SELL dominant
✓ EMA 9 < EMA 21 ← align SELL
✓ RSI < 50       ← align SELL
✓ H4 Trend DN  ← align SELL
✗ CI Uptrend (57) ← CI ขึ้น แต่ dominant ขาลง
✗ VWAP Uptrend  ← VWAP ขึ้น แต่ขัด SELL
💡 กฎอ่านง่าย ทุกตัวสีเดียวกัน = Perfect Setup — รอให้ทุก ✓ เป็นสีเดียวกันหมด (เขียวทั้งหมด = BUY, แดงทั้งหมด = SELL) ก่อนเข้าเทรดจะได้ความน่าเชื่อถือสูงสุด
SECTION 06
Risk Presets — ตั้งค่าความเสี่ยง
เลือกระดับความเสี่ยงที่ Settings → IRT GOLD Settings → Risk Preset

🛡️ Conservative

SL: 2.5× ATR
TP: 2.0× SL

เหมาะสำหรับผู้ที่ต้องการความปลอดภัยสูง ยอมรับ RR ที่น้อยกว่าเพื่อ SL ที่กว้าง

⚖️ Balanced

SL: 1.5× ATR
TP: 2.0× SL

ค่าเริ่มต้น — สมดุลระหว่างความเสี่ยงและผลตอบแทน เหมาะกับการเทรดทั่วไป

⚡ Aggressive

SL: 1.0× ATR
TP: 2.5× SL

SL แคบ TP ไกล — RR สูง แต่โดน SL ได้บ่อยกว่า เหมาะกับ Trend Strong

⚡ Scalping

SL: 0.8× ATR
TP: 1.5× SL

สำหรับ Scalp TF ต่ำ SL และ TP แคบ เข้าออกเร็ว เหมาะกับ M5-M15

วิธีตั้งค่าและเริ่มใช้งาน
ขั้นตอนการ setup ตั้งแต่ต้น
1

Add Indicator to Chart

เปิด Pine Script Editor บน TradingView → วาง Code → กด Add to Chart

2

เปิด Dashboard Panel

คลิก Settings (⚙️) → หัวข้อ "━━ IRT GOLD Settings ━━" → ติ๊ก ✓ Dashboard Panel

3

เลือก Risk Preset

ที่ Settings → Risk Preset เลือก Conservative / Balanced / Aggressive / Scalping ตามสไตล์การเทรด

4

อ่าน Dashboard ก่อนเทรด

ตรวจสอบ Score และ Checklist — รอให้เงื่อนไขส่วนใหญ่เป็นสีเดียวกัน (เขียวหรือแดง) ก่อนเข้าเทรด

5

รอสัญญาณ IRT B / IRT S

เมื่อ Dashboard ยืนยันทิศทาง รอสัญญาณ IRT B (BUY) หรือ IRT S (SELL) เข้ามา พร้อมกล่อง SL/TP อัตโนมัติ

6

ใช้โซน SUPPLY/DEMAND ประกอบ

สังเกต MSZ Zone ใกล้เคียง — โซน AAA ขึ้นไปที่ราคาใกล้อยู่ช่วยยืนยันจุดกลับตัวได้ดี

⚠️ ข้อควรระวัง หาก Dashboard แสดง ✗ สีส้ม จำนวนมาก หรือ CI แสดง Sideway — งดเทรด รอให้ตลาดเลือกทิศทางชัดเจนก่อน
